# Бинарный поиск

```
int bin search(int arr[], int size) {
    int left = 0;
    int right = size-1;
    int mid = (right+left)/2;
    while(n != arr[mid]){
        mid = (left+right)/2;
        if(n<arr[mid]){
            right = mid;
        }else if(n>arr[mid]){
            left = mid;
        }
        if(left==right){
            return(-1);
        }
    }
    return(mid);
}
```

O(log n) - работает быстрее обычного поиска.
